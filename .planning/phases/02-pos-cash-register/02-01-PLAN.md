    **1. Create `src/components/pos/pos-register.tsx`** (new file):

    This is the main 'use client' component for the POS cash register. It receives all data as props and manages cart state via useReducer.

    Props interface:
    ```typescript
    interface PosRegisterProps {
      gymPasses: GymPass[]
      fnbProducts: FnbProduct[]
      clients: Client[]
      bungalows: Bungalow[]
    }
    ```

    Cart state types (define in this file, above the component):
    ```typescript
    interface CartItem {
      produitId: string
      nom: string
      prixUnitaire: number
      quantite: number
      type: 'gym-pass' | 'fnb'
    }

    interface CartState {
      items: CartItem[]
      client: Client | null
      isBungalowResident: boolean
    }

    type CartAction =
      | { type: 'ADD_ITEM'; payload: Omit<CartItem, 'quantite'> }
      | { type: 'REMOVE_ITEM'; payload: { produitId: string } }
      | { type: 'UPDATE_QUANTITY'; payload: { produitId: string; quantite: number } }
      | { type: 'SET_CLIENT'; payload: { client: Client; isBungalowResident: boolean } }
      | { type: 'CLEAR_CART' }
    ```

    Implement `cartReducer` function:
    - ADD_ITEM: If item already exists (same produitId), increment quantite by 1. Otherwise add with quantite: 1.
    - REMOVE_ITEM: Filter out item by produitId.
    - UPDATE_QUANTITY: If quantite <= 0, remove item. Otherwise update quantite.
    - SET_CLIENT: Set client and isBungalowResident.
    - CLEAR_CART: Reset to initial state (empty items, null client, false isBungalowResident).

    Initial state: `{ items: [], client: null, isBungalowResident: false }`

    Component layout:
    ```tsx
    <div className="grid grid-cols-[1fr_320px] h-full">
      {/* Left panel: Product area (tabs + grids) */}
      <div className="flex flex-col overflow-hidden border-r border-border">
        <p className="text-muted-foreground p-4">Product grids coming in Plan 02</p>
      </div>

      {/* Right panel: Cart sidebar */}
      <div className="flex flex-col bg-card">
        <p className="text-muted-foreground p-4">Cart sidebar coming in Plan 04</p>
      </div>
    </div>
    ```

    Export the CartItem, CartState, and CartAction types (they will be imported by child components in later plans).
    Export the PosRegister component as a named export.
