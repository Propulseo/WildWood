  <files>src/app/(admin)/clients/[id]/page.tsx</files>
  <action>
Create `src/app/(admin)/clients/[id]/page.tsx` as a Client Component ('use client').

**Data loading:**
- Use `useParams<{ id: string }>()` from 'next/navigation' to get client ID
- Load client and bungalows via `useEffect` calling `getClientById(id)` and `getBungalows()` from `@/lib/data-access`
- Get transactions from `useTransactions()` context
- `useState<Client | null>(null)` for client, `useState<Bungalow[]>([])` for bungalows
- Show a loading state while client is null (simple "Chargement..." text or return null)

**Compute client stats (useMemo):**
- Filter transactions by `t.clientId === client.id`
- `montantTotal`: sum of all transaction totals using `.reduce((sum, t) => sum + t.total, 0)`
- `datesVisite`: unique visit dates extracted from transactions -- `[...new Set(clientTxns.map(t => t.date.split('T')[0]))]`, sorted descending
- `nombreVisites`: `datesVisite.length` (or use `client.nombreVisites` from the data)
- `historiqueAchats`: all client transactions sorted by date descending

**Bungalow resident detection (useMemo):**
- If `!client.bungalowId`, not a resident
- Find the bungalow matching `client.bungalowId`
- Check if any reservation has `statut === 'en-cours'`
- Returns boolean `isResident`

**Page layout:**
- Back button at top: `ArrowLeft` icon + "Retour aux clients" linking to `/clients` via `router.push('/clients')` or `Link`
- Client header section (flex row):
  - Left: Avatar placeholder (first letter of prenom + nom in a circle, use a div with `bg-wildwood-bois text-white rounded-full w-16 h-16 flex items-center justify-center font-display text-2xl font-bold`)
  - Right: `prenom nom` as h1 (font-display text-3xl font-bold), email and telephone below
  - Badge "Resident bungalow" if isResident -- use Badge component with `bg-wildwood-bois text-white` and Home icon (lucide-react, h-3 w-3)

**Stats cards (grid of 3 cards):**
- Card 1: "Montant total depense" -- `{montantTotal.toLocaleString()} THB` in text-2xl font-bold
- Card 2: "Nombre de visites" -- `{client.nombreVisites}` in text-2xl font-bold
- Card 3: "Derniere visite" -- formatted date `format(parseISO(client.derniereVisite), 'dd/MM/yyyy')` or '-' if none

**Purchase history table:**
- Section heading: "Historique des achats" (h2, font-display text-xl font-bold)
- Use a simple table (can use standard HTML table with Tailwind or shadcn Table if imported)
- Columns: Date, Type, Articles, Montant
- For each transaction in historiqueAchats:
  - Date: `format(parseISO(t.date), 'dd/MM/yyyy HH:mm')`
  - Type: Badge with type label -- 'gym-pass' -> 'Gym', 'fnb' -> 'F&B', 'bungalow' -> 'Bungalow'
  - Articles: `t.items.map(i => i.nom).join(', ')`
  - Montant: `{t.total.toLocaleString()} THB` (text-right)
- If no transactions: show "Aucun achat enregistre" message

**Visit dates section:**
- Section heading: "Dates de visite" (h2, font-display text-xl font-bold)
- Display as a horizontal flex wrap of Badge components showing each unique date
- Each badge: `format(parseISO(date), 'dd/MM/yyyy')`

**Imports needed:**
