- Center: month name in French using `format(currentMonth, 'MMMM yyyy', { locale: fr })` -- capitalize first letter

**Calendar grid (CSS grid):**
- Container: `overflow-x-auto` for horizontal scrolling on smaller screens
- Grid style: `gridTemplateColumns: '120px repeat(${daysInMonth}, minmax(32px, 1fr))'`
- Grid style: `gridTemplateRows: '40px repeat(8, 56px)'`
- Compute `monthStart = startOfMonth(currentMonth)`, `monthEnd = endOfMonth(currentMonth)`, `daysInMonth = getDaysInMonth(currentMonth)`, `days = eachDayOfInterval({ start: monthStart, end: monthEnd })`

**Header row (day numbers):**
- First cell: empty (bungalow label column)
- Then one cell per day: day number (1-31), use `format(day, 'd')`
- Highlight today's column with a distinct background (e.g., `bg-wildwood-orange/10`)
- Weekend days (Sat/Sun) get `text-muted-foreground/60` styling
- Show abbreviated day name below number: `format(day, 'EEE', { locale: fr })` (lun, mar, mer...)

**Bungalow rows (8 rows):**
- First cell in each row: bungalow label `Bungalow ${numero}` with `font-medium text-sm`
- Each row is position: relative within the CSS grid
- Reservation bars use absolute grid positioning via `gridColumn: '${start} / ${end}'` and `gridRow`

**Reservation bars:**
- Helper function `getReservationColumns(res: Reservation)` returns `{ start, end } | null`:
  - Parse `res.dateDebut` and `res.dateFin` with `parseISO`
  - If reservation has no overlap with current month (resEnd < monthStart or resStart > monthEnd), return null
  - Clamp: `start = resStart < monthStart ? 1 : resStart.getDate()`
  - Clamp: `end = resEnd > monthEnd ? daysInMonth + 1 : resEnd.getDate() + 1`
  - Return `{ start: start + 1, end: end + 1 }` (+1 for bungalow label column offset)

- Status color map:
  - `'en-cours'`: `'bg-wildwood-lime text-white'`
  - `'confirmee'`: `'bg-wildwood-orange text-white'`
  - `'terminee'`: `'bg-muted text-muted-foreground'`
  - `'annulee'`: `'bg-destructive/30 text-destructive line-through'`

- Each bar renders: `{clientName} - {nuits}n - {montant.toLocaleString()} THB`
- Bar styling: `rounded-md px-2 py-1 text-xs truncate h-[40px] flex items-center`
- For `en-cours` reservations, prepend a `Dumbbell` icon (lucide-react, h-3 w-3 shrink-0) with title="Acces gym inclus" (BUNG-04)

**Grid row assignment for reservation bars:**
- Each bungalow has `numero` (1-8). The header row is row 1, so bungalow N occupies grid row `N + 1`.
- Reservation bars are rendered inside the grid with explicit `gridRow` and `gridColumn` style.

**Important implementation notes:**
- Reservation bars must be placed as direct children of the grid container with explicit row/column positioning, NOT nested inside bungalow row divs. This is because CSS grid child positioning requires direct children.
- Alternatively, use a subgrid approach: each bungalow row is a div spanning all columns, then reservation bars inside use absolute positioning. The simpler approach is to render ALL cells and bars as flat children of the grid container.
- The recommended approach: Render the grid as one flat container. First render header cells (row 1), then for each bungalow render the label cell (column 1) and day background cells, then render reservation bars on top using the same grid with explicit row/column.

  <action>
Add occupancy rate display to the bungalow calendar page (same file as Task 1).

**Monthly occupancy rate (useMemo):**
- `totalSlots = 8 * daysInMonth` (8 bungalows x days in displayed month)
- For each bungalow, for each reservation (excluding `statut === 'annulee'`):
  - Compute overlap with current month: `overlapStart = max(resStart, monthStart)`, `overlapEnd = min(resEnd, monthEnd)`
  - If overlap exists (`overlapStart <= overlapEnd`), count days with `eachDayOfInterval({ start: overlapStart, end: overlapEnd }).length`
  - Sum all occupied days across all bungalows
- `occupancyRate = Math.round((occupiedSlots / totalSlots) * 100)`

**Weekly occupancy breakdown (useMemo):**
- Use `eachWeekOfInterval({ start: monthStart, end: monthEnd }, { weekStartsOn: 1 })` to get week start dates (Monday start)
- For each week:
  - Clamp week boundaries to month: `weekStart = max(weekStartDate, monthStart)`, `weekEnd = min(endOfWeek(weekStartDate, { weekStartsOn: 1 }), monthEnd)`
  - `daysInWeek = eachDayOfInterval({ start: weekStart, end: weekEnd }).length`
  - `totalSlots = 8 * daysInWeek`
  - Count occupied slots same as monthly (iterate bungalows, reservations, compute overlap with week range)
  - Return `{ label: 'S' + format(weekStartDate, 'w'), rate: Math.round(occupied / totalSlots * 100) }`

**Display below the calendar grid:**
- Card with CardHeader "Taux d'occupation" and CardContent showing:
  - Monthly rate as large number: `{occupancyRate}%` in `text-3xl font-bold font-display text-wildwood-bois`
  - Subtitle: `{occupiedSlots} / {totalSlots} nuits-bungalow`
- Weekly breakdown as horizontal bar chart using Tailwind:
  - Each week: label (S1, S2...) + percentage + colored bar (`bg-wildwood-lime` width proportional to rate)
  - Use `style={{ width: '${rate}%' }}` for the bar fill
  - Display rate number at end of bar

**Layout:**
