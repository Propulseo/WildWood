<task type="auto">
  <name>Task 2: Build client list page with search, filters, and pagination</name>
  <files>src/app/(admin)/clients/page.tsx</files>
  <action>
Create `src/app/(admin)/clients/page.tsx` as a Client Component ('use client').

**Data loading:**
- Load clients and bungalows via `useEffect` calling `getClients()` and `getBungalows()` from `@/lib/data-access`
- Get transactions from `useTransactions()` context

**Derive "type de pass" per client (useMemo):**
- For each client, find all transactions where `t.clientId === client.id && t.type === 'gym-pass'`
- Sort by date descending, take the first one
- Extract `items[0].nom` as the pass type name
- If no gym-pass transaction, show `'-'`
- Extend client with computed `typePass` field

**Search state:**
- `useState('')` for search text
- Filter by `prenom + nom`, `email`, or `telephone` using `.toLowerCase().includes(q)`
- No debounce (35 items is instant)

**Filter state:**
- `useState('tous')` for `filterPass` -- Select dropdown with dynamically derived pass type options from transaction data (useMemo to get unique pass names)
- `useState('tous')` for `filterPeriode` -- Select dropdown with options: "Toutes les periodes", "7 derniers jours", "30 derniers jours", "90 derniers jours", "Cette annee"
- Period filter: compare client's `derniereVisite` with `subDays(new Date(), N)` using `isAfter` from date-fns. For "Cette annee", use `startOfYear(new Date())`.

**Filtered results (useMemo):**
- Apply search filter, then pass type filter, then period filter
- Chain all three filters in sequence

**Pagination:**
- `ITEMS_PER_PAGE = 10`
- `useState(1)` for current page
- `totalPages = Math.ceil(filtered.length / ITEMS_PER_PAGE)`
- `paginated = filtered.slice((page - 1) * ITEMS_PER_PAGE, page * ITEMS_PER_PAGE)`
- `useEffect` resets page to 1 when search, filterPass, or filterPeriode changes

**Table layout using shadcn Table:**
- Page header: "Clients" (h1 font-display text-3xl font-bold) + subtitle "35 clients" showing filtered count
- Search bar: Input component with Search icon (lucide-react), placeholder "Rechercher par nom, email ou telephone..."
- Filters row: Two Select dropdowns side by side -- "Type de pass" and "Periode"
- Table columns: Nom, Prenom, Email, Telephone, Type de pass, Derniere visite, Nb visites
- Table rows: `cursor-pointer hover:bg-muted/50`, onClick navigates to `/clients/${client.id}` via `useRouter` from `next/navigation`
- Format `derniereVisite` with `format(parseISO(date), 'dd/MM/yyyy')` from date-fns
- Show `'-'` for missing email/telephone

**Pagination controls:**
- Below table: "Page X sur Y" text + Previous/Next buttons using Button component (variant="outline", size="sm")
- Previous button disabled when page === 1
- Next button disabled when page === totalPages
- Showing "X - Y sur Z clients" count

**Imports needed:**
